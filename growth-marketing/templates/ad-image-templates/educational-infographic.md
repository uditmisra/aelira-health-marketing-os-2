# Educational Infographic

## Best For
- business_model: b2b_saas | healthcare_services | professional_services
- stage: awareness | consideration
- platforms: LinkedIn Single Image, Meta Feed, Google Display (rectangle)
- when to use: When your audience needs to understand a process, a comparison, or a framework before they can value your product — and when that understanding can be delivered in a single, scannable visual. Effective when the audience self-identifies as analytical or detail-oriented (legal teams, ops leaders, medical professionals, financial buyers) and skips ads that feel like ads.

## Visual Layout
```
┌─────────────────────────────────────┐
│  {{headline}}                       │  ← full-width top, 10% frame height
│  (question or framework name)       │     light bg, dark text or reverse
├─────────────────────────────────────┤
│                                     │
│  [main diagram zone]                │
│                                     │
│  OPTION A: numbered steps flow      │
│  ┌───┐    ┌───┐    ┌───┐    ┌───┐  │
│  │ 1 │ →  │ 2 │ →  │ 3 │ →  │ 4 │  │  ← 60% frame height
│  └───┘    └───┘    └───┘    └───┘  │
│  label    label    label    label   │
│                                     │
│  OPTION B: comparison table         │
│  ┌──────────────┬─────────────┐     │
│  │  Without X   │   With X    │     │
│  ├──────────────┼─────────────┤     │
│  │  row 1       │   row 1     │     │
│  │  row 2       │   row 2     │     │
│  └──────────────┴─────────────┘     │
│                                     │
├─────────────────────────────────────┤
│  {{body_copy}} (source/credential)  │  ← footer strip, 8% frame height
│  [logo]              [ {{cta}} ]    │
└─────────────────────────────────────┘
```

Zones:
- Top bar: full-width, 10-12% frame height. White or very light gray background, dark text. Contains the headline as a question or framework label — "How contract review actually works" or "The 4 stages of contract risk."
- Main diagram zone: 68-72% of frame. The primary visual is either: (A) a numbered-step flow (4-5 horizontal steps with connecting arrows), (B) a 2-column comparison table, or (C) a labeled hierarchy/pyramid. Brand primary color used for active/positive elements. Gray used for inactive/negative states. All labels in dark gray body text, small, weight 500.
- Footer strip: full-width, very light gray or white, thin top border. Contains body copy (source citation or credential) on the left, logo center-left, CTA button on the right.

```
FOOTER (8% frame height): very light gray strip with thin top rule. Left side: "{{body_copy}}" in small gray text (source or credential). Right side: brand logo at small size. Far right: CTA button reading "{{cta}}" — brand primary fill, white text, 4px rounded corners, weight 600.

Negative prompt: clip art icons, 3D charts, pie charts, decorative stars, speech bubbles, stock photography integrated into diagram, serif fonts, multiple competing accent colors, rainbow color schemes, drop shadows on table rows.
```

## Headline Guidance
- Character limit: 60 chars
- Style: A question the target audience would ask themselves ("How long should contract review actually take?") or a framework title they want to understand ("The 4 contracts that carry the most risk"). Analytical, curious framing. Not benefit-first — insight-first.

## Body Copy Guidance
- Character limit: 70 chars
- Style: Source attribution or credential statement — "Based on 500+ legal team surveys, SpotDraft 2025" or "From the 2025 In-House Legal Benchmark Report." Legitimizes the diagram. Can be blank if the diagram is self-evidently internally sourced.

## CTA Guidance
- Style: Learning-forward, not sales-forward — "GET THE FULL REPORT", "DOWNLOAD THE GUIDE", "SEE THE BENCHMARK DATA". Avoid "BOOK A DEMO" as first CTA in this template — the audience is still in learning mode. Button, right-aligned in footer, brand primary.
